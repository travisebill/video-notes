**主講｜陳文茜 / TVBS 文茜的世界財經周報 / 2026年5月10日**

## 一、主題與背景

本集為《文茜財經周報》完整版下集，主題圍繞 2026 年美國對伊朗軍事行動所引發的全球金融體系連鎖效應。核心論點是：美國對伊朗的戰爭與金融制裁，意外加速了「去美元化」趨勢，反而扶植了中國「石油人民幣」的崛起。節目同時追蹤 AI 代理人（Agent）時代帶動的半導體需求大洗牌，以及台韓股市創史詩級里程碑、Intel 驚人反彈等議題。

## 二、章節脈絡

### Section 1｜石油美元的完美風暴：伊朗戰爭加速去美元化（00:00 ~ 10:00）

**重點摘要：** 戰爭拖久產生的輻射效應，讓美元從伊朗戰爭中成為受害者，石油人民幣趁勢崛起。

**內容：**
- 1973–74 年美國與 OPEC 談判，石油美元結算體系確立，奠定美元世界貨幣地位
- 美國對伊朗的金融制裁 → 伊朗對中國石油輸出採人民幣結算
- 對俄羅斯石油制裁 → 部分解除後對歐洲輸出也用人民幣結算
- 「美國沒有意思要扶植人民幣走強國際化，但它卻做到了——它的仇家想做都做不到的事」
- 德意志銀行報告：伊朗戰爭正演變成石油美元體系的「完美風暴」

> 「美國蠢到了極點。中國什麼都沒做，就是美國人一路的幫忙。」

### Section 2｜荷姆茲封鎖與人民幣結算機制（10:00 ~ 18:00）

**重點摘要：** 伊朗封鎖荷姆茲海峽後，允許少數油輪通行，條件是必須以人民幣結算。

**內容：**
- 伊朗允許少數油輪通過荷姆茲海峽，條件：石油必須以人民幣交易（非美元）
- 《財富中文網》：截至 3 月，中東與中國石油貿易中，人民幣結算佔比飆升至 **41%**，首次超越歐元
- 美元在中東石油結算佔比從 90%+ 降至 **52%**
- 伊朗近九成原油出口至中國，透過人民幣交易
- 金磚國家正日益繞過美元進行能源交易

> 「伊朗非常認真地使用人民幣以迴避美國制裁。」 — 前 IMF 首席經濟學家 Kenneth Rogoff

### Section 3｜SWIFT 替代方案：CIPS 與 mBridge 崛起（18:00 ~ 22:00）

**重點摘要：** 伊朗與俄羅斯被踢出 SWIFT，中國 CIPS 成為主要替代清算系統。

**內容：**
- 伊朗與俄羅斯雙雙被踢出 SWIFT（全球銀行互聯基礎設施）
- 中國人民銀行開發的 **CIPS**（人民幣跨境支付系統）成為主要替代方案
- 新加坡《聯合早報》：3 月 CIPS 日均交易量從 6,600 億增至 **9,000 多億人民幣**
- 阿聯退出 OPEC，同時是中國支持的 **mBridge** 支付系統創始成員（基於區塊鏈的 CBDC）
- mBridge 規模遠小於 CIPS，但有潛力成為 SWIFT 替代方案

### Section 4｜全球外匯儲備與黃金：去美元化的數據證據（22:00 ~ 28:00）

**重點摘要：** IMF 報告顯示美元外匯儲備佔比跌至 30 年低位，各國央行加速增持黃金。

**內容：**

| 指標 | 數據 |
|------|------|
| 美元外匯儲備佔比（2025 Q4） | **56.77%**（30 年低位） |
| 1999 年美元佔比 | 超過 71% |
| 美元指數本周 | 跌 0.38% |
| 黃金期貨本周 | 漲 2.03%，$4,723.7 |
| 白銀期貨本周 | 漲 6.43% |

**2026 年各國央行增持黃金排名：**
1. 美國（不需黃金也支撐美元）
2. 義大利（增持 245 噸，達 451.84 噸）
3. **中國**（從第 5-6 名躍升至第 3，加速去美元）
4. 俄羅斯
5. 印度
6. 日本
7. 波蘭
8. 土耳其
9. 烏茲別克
10. 哈薩克

> 中國央行已連續 18 個月增持黃金。

### Section 5｜美國國債危機：利息超越軍事支出（28:00 ~ 32:00）

**重點摘要：** 美國國債首度超過一年 GDP，利息成本已超越國防與醫療支出。

**內容：**
- 美國國債累計 **39 兆美元**，高於 GDP 的 31.22 兆美元（二戰以來首見）
- 利息成本已超過國防預算與醫療支出
- 長期支出高於稅收 + 社會安全/醫療保險福利增加 → 持續舉債
- 專家警告可能引發主權債務危機（借貸成本過高、投資人要求更高利率）
- 川普政府缺少整體減債改革計畫，戰爭+關稅加劇市場信心動搖

### Section 6｜AI 代理人時代：半導體需求大洗牌（32:00 ~ 38:00）

**重點摘要：** AI 無法單靠 GPU，需要 CPU 與記憶體協作，帶動半導體板塊全面爆發。

**內容：**

| 趨勢 | 說明 |
|------|------|
| GPU → CPU 轉移 | AI Agent 時代每人可有 2-10 個 Agent，需要更多 CPU 計算 |
| 記憶體需求暴增 | HBM（高頻寬記憶體）供不應求，美光市值突破 8,000 億美元 |
| 台股創歷史新高 | 42,156 點，總市值 4.47 兆美元 → 全球第六大 |
| 韓股破 7,000 點 | 三星市值突破 1 兆美元，SK 海力士年漲 144% → 全球第七大 |

> 「你可以有 2、3、5、10 個 Agent，這些 Agent 在我們所有工作上為我們工作，代表你需要更多的 CPU 計算。」 — AMD CEO 蘇姿丰

### Section 7｜Intel 驚人反彈：年漲 238% 的背後（38:00 ~ 44:00）

**重點摘要：** Intel 做出正確策略決策，加上國家級支持，成為 2026 年最驚人的半導體復甦故事。

**內容：**

**Intel 2026 年迄今漲幅：238.54%**（NASDAQ 55 年來最佳單月表現，4 月漲 114%）

**關鍵策略：**
- 回購愛爾蘭 Fab34 晶圓廠股權（生產效率遠高於 Arizona）
- 先進封裝技術 + 生態系聯盟（聯發科、聯電、世界先進）
- 美國政府前所未有入股 9.9%

**核心客戶排隊接手 Intel 代工產能：**
- NVIDIA（投資 50 億美元入股）
- Microsoft、Amazon、Google、Meta、Qualcomm
- Tesla、SpaceX、xAI（加入 Terraflop 計畫）
- Apple（尋求替代台積電的晶片供應）

> 對美國企業而言，長期依賴台積電在台海危機下不穩妥，依賴三星不如扶植 Intel。

### Section 8｜敘利亞：戰火中的油罐車經濟（44:00 ~ 50:00）

**重點摘要：** 荷姆茲海峽封鎖後，伊拉克透過敘利亞用油罐車陸路運油，為經歷 14 年內戰的敘利亞帶來重生契機。

**內容：**

- 伊拉克 4/1 重啟與敘利亞的邊境口岸，用油罐車運油
- 1 艘油輪 ≈ 1 萬台油罐車，從伊拉克到敘利亞港口需 12 天、1,200 公里
- 敘利亞總統沙拉表態不加入戰事任何一方，只對油罐車運輸有興趣
- 敘利亞石油公司評估：近期伊拉克石油輸送量可達每月 50 萬噸
- 但專家指出，每日 8 萬桶的陸運極限遠不足以替代海運 → 這是「騷擾」而非解決方案

### Section 9｜俄羅斯：《大師與瑪格麗特》與停火的虛偽（50:00 ~ 55:00）

**重點摘要：** 布爾加科夫諷刺小說改編電影在俄羅斯意外上映，觀眾從中看到對普丁政權的批判；俄烏停火協議再成「詐欺」。

## 三、關鍵概念定義

| 概念 | 定義 | 背景 |
|------|------|------|
| **石油美元（Petrodollar）** | 全球石油以美元計價與結算的體系 | 1974 年美國與沙烏地協議確立 |
| **石油人民幣（Petroyuan）** | 以人民幣取代美元進行石油定價與交易 | 2013 年上海原油期貨起逐步發展 |
| **CIPS** | 人民幣跨境支付系統，中國人民銀行開發 | SWIFT 被制裁國的主要替代方案 |
| **mBridge** | 基於區塊鏈的多國 CBDC 支付系統 | 阿聯為創始成員，潛在 SWIFT 替代方案 |
| **AI Agent（AI 代理人）** | 能自主執行多步驟任務的 AI 系統 | 2026 年核心趨勢，需要大量 CPU + 記憶體 |
| **HBM（高頻寬記憶體）** | AI/GPU 用高速記憶體 | 美光、SK 海力士、三星主力產品 |
| **Terraflop 計畫** | 馬斯克旗下次世代晶片製造計畫 | Intel 加入，推進先進製程 |
| **SWIFT** | 全球銀行金融通訊系統 | 將俄/伊踢出被稱為「金融核彈」 |

## 五、核心主旨

> 美國對伊朗的戰爭意外成為「石油人民幣」崛起的最大推手——去美元化趨勢因戰爭加速，全球金融從單極走向多極；同時 AI 代理人時代重塑半導體需求版圖，CPU 與記憶體取代 GPU 成為新的戰略稀缺資源。
