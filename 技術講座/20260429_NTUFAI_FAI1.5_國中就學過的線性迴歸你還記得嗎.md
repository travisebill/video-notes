## 一、主題與背景

FAI 1.5「國中就學過的線性迴歸你還記得嗎」是林軒田教授 Machine Learning 章節的第五講，約 18.5 分鐘。本講正式進入 Linear Regression 的詳細推導——從「申請工作」範例延伸，把 Binary Classification（適合 / 不適合）升級為 Regression（給申請者一個評分排名）。

Linear Regression 與 Linear Classification 的關鍵差異：Classification 的 Output 是 binary（yes / no），用 Sign function 切兩邊；Regression 的 Output 是連續值（0~100 分），直接用 $\mathbf{w}^T \mathbf{x}$ 輸出。林軒田教授指出，這個差異讓 Linear Regression 可以用 Squared Error 當作 Error Measure，並且有 Closed-form Solution——不需要疊代學習，一步就能算出最佳 $\mathbf{w}^*$。

本講的另一條主軸是 **ML 與統計學的關係**。Linear Regression 是國中數學的最小平方法、統計學的線性模型、也是 ML 的基礎——三個領域在這個模型上交會。林軒田教授強調：統計學提供強大的數學工具，ML 把它們應用到實際的 prediction 任務上；兩者相輔相成。統計重視 inference（推論 $\mathbf{w}$ 的信賴區間），ML 重視 prediction（在 unseen data 上表現好）。

**核心議題一句話**：Linear Regression 把 Binary Classification 升級為連續值預測，可以用 Squared Error 量化誤差，並且有 Closed-form Solution 直接解出 $\mathbf{w}^*$——這是 ML 中少數「一步到位」的模型，也是統計、數學、ML 三個領域交會的經典案例。

## 二、章節脈絡

### Section 1｜從 Classification 到 Regression（00:00 ~ 03:00）

**重點摘要：** Classification 輸出 binary；Regression 輸出連續值；Linear Model 兩者通用。

**內容：**
- FAI 1.1-1.4 用 Linear Classification 為例
- Classification 範例：給申請者履歷，輸出適合 / 不適合（binary）
- Regression 範例：給申請者履歷，輸出一個 score（例如 0~100 分）
- Linear Model 兩者通用：$\mathbf{w}^T \mathbf{x}$ 是 score；sign($\mathbf{w}^T \mathbf{x}$) 是 classification
- Linear Regression 在工業界最常用：預測股價、預測銷售、預測房價、預測點擊率

> "如果是 Linear Regression 的話它的 Output 不是 yes or no，它可能是我想 output 你根據這個 candidate 我的 overall 這些 score 應該是比較前面比較後面…Output 可能就從剛剛的 positive 跟 negative 改成是一個數值。"

### Section 2｜Error Measure：Squared Error（03:00 ~ 06:00）

**重點摘要：** Regression 用 Squared Error；Pointwise 平方加起來平均。

**內容：**
- Linear Classification 用 0/1 Error（錯一個算一個）
- Linear Regression 用 Squared Error：(預測值 - 真實值)²
- Squared Error 的優點：數學上可微分、有 Closed-form Solution、懲罰大錯（平方放大差距）
- 為什麼是 Squared Error 而不是 Absolute Error？因為 squared 可微、好處理
- Pointwise Error：對每個樣本算 error，總和 / 平均
- In-Sample Error $E_{in}$ = average squared error on training data
- 目標：minimize $E_{in}$ 找出 $\mathbf{w}^*$

> "如果是 regression 的話，我們就會去看說我們預測出來的值跟它真值的差異，所以常常會用 Square Error 來表示。"

### Section 3｜Closed-form Solution：矩陣微分（06:00 ~ 10:00）

**重點摘要：** Linear Regression 有 Closed-form Solution $\mathbf{w}^* = (X^T X)^{-1} X^T \mathbf{y}$；不需要 Learning Algorithm 疊代。

**內容：**
- Linear Regression 是 ML 中少數有 Closed-form Solution 的模型
- 把 $E_{in}$ 寫成 $\mathbf{w}$ 的二次式，對 $\mathbf{w}$ 微分等於零
- 解出：$\mathbf{w}^* = (X^T X)^{-1} X^T \mathbf{y}$（pseudo-inverse 形式）
- 數學步驟：
  1. 把 $N$ 個樣本的 $(\mathbf{x}_n, y_n)$ 排成矩陣 $X$（$N \times d$）和 $\mathbf{y}$（$N \times 1$）
  2. $E_{in}(\mathbf{w}) = \frac{1}{N} \|X\mathbf{w} - \mathbf{y}\|^2$
  3. $\nabla E_{in} = \frac{2}{N} X^T (X\mathbf{w} - \mathbf{y}) = 0$
  4. 解得 $\mathbf{w}^* = (X^T X)^{-1} X^T \mathbf{y}$
- 優點：一步到位，不需要 PLA 那樣疊代
- 缺點：$X^T X$ 可能 singular（當 features linearly dependent 時），需要 pseudo-inverse
- 這也是統計學中最小平方法的標準解

> "Linear Regression 其實是可以用一個 Close-form Solution…有一個非常複雜的一個線性代數就可以把這個 $\mathbf{w}$ 直接解出來。"

### Section 4｜Linear Regression 與 Linear Classification 的關係（10:00 ~ 14:00）

**重點摘要：** 兩者 Hypotheses Set 一樣（都是 $\mathbf{w}^T \mathbf{x}$）；差別在 Error Measure 與最後切分方式。

**內容：**
- 兩者 Hypotheses Set 都是所有 hyperplane 集合
- Linear Classification：sign($\mathbf{w}^T \mathbf{x}$) 切兩邊，用 0/1 Error
- Linear Regression：$\mathbf{w}^T \mathbf{x}$ 直接輸出，用 Squared Error
- 注意：用 Squared Error 訓練 Regression 後的 $\mathbf{w}$ 也可以用在 Classification（做 sign）
- 這個 connection 顯示：兩者其實是同一個 Hypotheses Set 下的不同任務
- Linear Regression 的 Closed-form Solution 比 PLA（Classification 的 Learning Algorithm）快很多
- 業界實務：很多 Classification 任務先用 Linear Regression 快速收斂到接近最佳解，再用 Logistic Regression 微調

> "這邊的 $\mathbf{w}$ 其實就是對應剛剛看到那個 $\mathbf{w}$，每一個維度的這個 $\mathbf{w}$，它就是這個 Linear Model 的 $g$。"

### Section 5｜ML 與統計的關係：相輔相成（14:00 ~ 18:24）

**重點摘要：** 統計重視 inference（信賴區間）；ML 重視 prediction（unseen data）；兩者用相同的數學工具。

**內容：**
- Linear Regression 是國中數學的最小平方法、統計學的線性模型、也是 ML 的基礎
- 統計學觀點：給定資料假設，找出 $\mathbf{w}$ 的估計與信賴區間，強調 inference
- ML 觀點：找 $g$ minimize $E_{out}$，強調 prediction
- 兩者用相同的數學工具（最小平方、線性代數、機率），但目標不同
- 統計學的 model assumption 嚴格（必須常態分布、獨立同分布）；ML 的假設較寬鬆
- ML 從統計學借用很多工具，但也發展出新的方法（如 SVM、Neural Network）
- 兩者關係：相輔相成，統計給 ML 數學基礎，ML 給統計真實應用場景

> "統計其實提供給 Machine Learning 非常多有用的工具，所以他們兩個算是相互相成的關係。"

## 三、關鍵概念定義表

| 概念 | 定義 | 出處 / 應用 |
|------|------|-------------|
| Linear Regression | $\mathbf{w}^T \mathbf{x}$ 輸出連續值的 Linear Model | Section 1 |
| Linear Classification | sign($\mathbf{w}^T \mathbf{x}$) 切 binary 的 Linear Model | FAI 1.1-1.4 |
| Squared Error | (預測 - 真實)²，regression 的標準 Error Measure | Section 2 |
| 0/1 Error | 錯一個算一個，classification 的 Error Measure | Section 2 |
| Closed-form Solution | 直接由數學公式算出最佳解，不需疊代 | Section 3 |
| Pseudo-inverse | $(X^T X)^{-1} X^T$，處理 $X^T X$ singular 的情況 | Section 3 |
| Singular Matrix | 行列式為零的矩陣，無法直接求 inverse | Section 3 |
| In-Sample Error $E_{in}$ | training data 上的 average error | Section 2 |
| Out-of-Sample Error $E_{out}$ | unseen test data 上的 average error | FAI 1.3 |
| Gradient $\nabla E_{in}$ | $E_{in}$ 對 $\mathbf{w}$ 的偏微分向量 | Section 3 |
| Hessian Matrix | $E_{in}$ 對 $\mathbf{w}$ 的二次微分矩陣 | Section 3 |
| Convex Function | 函數只有一個 global minimum | Section 3 |
| Pseudo-inverse 解法 | 數值穩定的 closed-form 替代方案 | Section 3 |
| Inference | 統計學強調的「$\mathbf{w}$ 估計的信賴區間」 | Section 5 |
| Prediction | ML 強調的「在 unseen data 上表現好」 | Section 5 |
| Ordinary Least Squares (OLS) | 統計學中的最小平方估計法 | Section 5 |
