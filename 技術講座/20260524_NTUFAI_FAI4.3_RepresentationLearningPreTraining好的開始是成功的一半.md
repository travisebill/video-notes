## 一、主題與背景

FAI 4.3「好的開始是成功的一半 — Representation Learning / Pre-Training」由陳縕儂教授切入深度學習訓練流程中最容易被低估的環節——好的起始點（initialization）對訓練成效的決定性影響，並由此展開「表徵學習」（Representation Learning）與「預訓練」（Pre-Training）的完整圖像。

節目開頭先回到上一講反向傳播的尾聲。深度學習的優化問題不是凸函數，從不同起始點出發，最終收斂到的局部最佳解可能截然不同。如果能選擇一個比較好的起始位置，訓練就更容易找到一個比較好的解。這個「好的開始是成功的一半」概念，正是表徵學習的核心動機——希望能學到一組具有代表性的特徵表示，作為下游任務的良好初始化基礎。

節目第一個介紹的代表技術是「自編碼器」（Autoencoder）。它的結構是一個瓶頸式的對稱網路：輸入經過編碼器壓縮到低維潛在空間，再由解碼器還原回原始維度。訓練目標是讓輸出與輸入越接近越好，迫使瓶頸層學到資料最核心的表徵。這種「無監督式學特徵」的方式，正是早期表徵學習的經典做法。

接著節目把視野拉高到預訓練的宏觀策略。現代語言模型如 BERT、GPT 系列，本質上都是在大規模語料上做預訓練的模型——可以把它們視為在語言這個領域中找到了一組非常具有代表性的特徵。預訓練完成後，這組特徵可以遷移到各種下游任務（情緒分析、問答、摘要等），只需要少量標註資料做微調（fine-tuning）就能達到很好的效果。這種「先學通用表徵、再針對任務微調」的兩階段流程，已成為深度學習在 NLP、影像、語音等領域的標準範式。

預訓練的成功也帶動了一系列衍生技術。資料增強（Data Augmentation）透過各種隨機變換擴充訓練資料，等同於隱式地讓模型學到更多視角的不變性；對比學習（Contrastive Learning）則透過「拉近相似樣本、推遠相異樣本」的方式，在無標註資料上學到極具鑑別度的表徵；自監督學習（Self-Supervised Learning）更進一步，讓資料本身產生監督訊號，徹底擺脫對人工標註的依賴。

**核心議題一句話**：好的初始化決定訓練能否成功，而表徵學習與預訓練正是工程化「找到好初始化」的系統性方法；從 autoencoder 到 BERT、GPT，這套方法論揭示了深度學習從「能訓練」到「訓練得好」的關鍵轉折。

## 二、章節脈絡

### Section 1｜為什麼好的開始很重要（00:00 ~ 06:00）

**重點摘要：** 深度學習優化是非凸的；不同起始點 → 不同局部最佳解；好的初始化讓訓練更容易收斂到好解；這是表徵學習的動機。

**內容：**
- 反向傳播給了有效率的計算方式，但優化本身仍然是非凸問題
- 從不同 initial point 出發，最終可能落到不同的 local optimal
- Loss surface 不是一個漂亮的曲面，可能是高低起伏的丘陵地形
- 好的起始點讓訓練「走對方向」，更容易到達好的局部最佳解
- 這個觀察衍生出表徵學習的研究路線
- 表徵學習的目標：學到一組有代表性的特徵表示，作為下游任務的好基礎

> "如果有一個比較好的開始，可能就比較容易繼續訓練到一個更好的地方。這樣的概念其實就衍生出了非常多Refsentation Learning的技術。"

### Section 2｜Autoencoder 與早期表徵學習（06:00 ~ 15:00）

**重點摘要：** Autoencoder 是瓶頸式對稱網路；編碼器壓縮、解碼器還原；瓶頸層被迫學到資料最核心的特徵；這是早期無監督表徵學習的代表技術。

**內容：**
- Autoencoder 結構：輸入 → 編碼器 → 瓶頸 → 解碼器 → 輸出
- 訓練目標：讓輸出 ≈ 輸入
- 瓶頸的維度遠小於輸入維度 → 必須學到壓縮表示
- 瓶頸層的輸出就是學到的特徵表徵
- Autoencoder 是「無監督式」學特徵的代表
- 後續衍生：去噪自編碼器（Denoising）、變分自編碼器（VAE）等

> "Refsentation Learning是什麼意思？就是我希望能夠學到一些比較具有代表性的資訊，那這些具有代表性的資訊比較能夠幫助我們更容易到達好的地方。"

### Section 3｜預訓練（Pre-Training）的策略（15:00 ~ 25:00）

**重點摘要：** 預訓練 = 在大量無標註資料上學通用表徵；下游任務只需要少量標註資料微調；BERT、GPT 都是這個範式的代表。

**內容：**
- 預訓練：在大型無標註資料集上學通用特徵
- 微調（Fine-Tuning）：用少量標註資料把通用特徵調整到特定任務
- 預訓練 + 微調的兩階段流程是 NLP、影像、語音的標準範式
- BERT：用遮罩語言模型做預訓練
- GPT：用自回歸語言模型做預訓練
- 預訓練成功的關鍵：資料量足夠大 + 模型容量足夠大

> "到現在大家看到這些語言模型，它其實就是一個 pre-tune的模型，所以它大致上你也可以把它想成是它其實找到了在語言的資訊當中一些非常具有代表性的 features。"

### Section 4｜資料增強與自監督學習（25:00 ~ 33:00）

**重點摘要：** 資料增強（Data Augmentation）擴充訓練資料；對比學習（Contrastive Learning）學鑑別度特徵；自監督學習（Self-Supervised Learning）讓資料自己產生監督訊號。

**內容：**
- 資料增強：旋轉、裁切、調色、雜訊等隨機變換擴充資料
- 等同於讓模型學到「不變性」
- 對比學習：拉近相似樣本，推遠相異樣本
- SimCLR、MoCo、CLIP 是這個方向的代表
- 自監督學習：設計 pretext task，讓資料自己產生監督
- 徹底擺脫對人工標註的依賴

### Section 5｜表徵學習與遷移學習的整合觀點（33:00 ~ 39:17）

**重點摘要：** 表徵學習的價值在跨任務、跨領域的可遷移性；學到一組好特徵可以重複用於多個下游任務；這是深度學習進入「大模型時代」的理論基礎。

**內容：**
- 一個好的預訓練模型 = 一組通用的、可遷移的特徵
- 不同任務、不同領域都可以從同一組特徵出發微調
- 這是「基礎模型」（Foundation Model）概念的雛形
- 預訓練的規模越大、特徵越通用，下游任務的表現越好
- 這也是 GPT-4、Claude 等大模型能跨任務表現的根本原因

## 三、關鍵概念定義

| 概念 | 定義 | 重要性 |
|------|------|--------|
| 局部最佳解（Local Optimal） | 比鄰近解都好但不是全域最佳的解 | 深度學習優化的常態 |
| 表徵學習（Representation Learning） | 自動學到資料的特徵表示，而非依賴人工設計 | 深度學習的核心能力 |
| 自編碼器（Autoencoder） | 瓶頸式對稱網路，用輸出還原輸入 | 早期表徵學習的代表技術 |
| 預訓練（Pre-Training） | 在大型資料集上學通用特徵，作為下游任務的起點 | 現代深度學習的標準範式 |
| 微調（Fine-Tuning） | 用少量標註資料把預訓練特徵調整到特定任務 | 預訓練 + 微調的兩階段流程 |
| 資料增強（Data Augmentation） | 用隨機變換擴充訓練資料 | 增加資料多樣性、提升泛化 |
| 對比學習（Contrastive Learning） | 拉近相似樣本、推遠相異樣本的學習方式 | 自監督表徵學習的代表方法 |
| 自監督學習（Self-Supervised Learning） | 設計 pretext task，讓資料自己產生監督訊號 | 不需人工標註就能學到好特徵 |

## 五、核心主旨

> 深度學習的優化問題是非凸的，從不同起始點出發可能落到截然不同的局部最佳解；好的初始化是訓練成功的關鍵前提。表徵學習（Representation Learning）的核心目標，就是學到一組具有代表性的特徵表示，作為下游任務的良好基礎——早期的 Autoencoder 透過瓶頸式對稱網路學壓縮表徵，現代的預訓練（Pre-Training）則透過 BERT、GPT 等大模型在大規模無標註資料上學通用特徵，再以少量標註資料微調（Fine-Tuning）適配到具體任務。這套「先學通用表徵、再針對任務微調」的兩階段流程，已成為深度學習在 NLP、影像、語音等領域的標準範式。延伸的資料增強（Data Augmentation）、對比學習（Contrastive Learning）、自監督學習（Self-Supervised Learning）則進一步擴展了「無需大量人工標註就能學到好特徵」的可能性，成為深度學習從「能訓練」走向「訓練得好」的關鍵轉折。
