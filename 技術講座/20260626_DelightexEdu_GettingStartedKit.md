## 二、章節脈絡

### Section 1｜平台概述：As easy as 1, 2, 3!

**重點摘要：** Delightex Edu 是一個免費的 3D / VR / AR 教育工具平台，整份入門手冊圍繞三個主步驟展開 — Sign up（註冊）、Create（創建專案）、Explore（探索作品）。在進入細節之前，平台已建立基本技術門檻：網頁版可在任何電腦（含 Google Chromebook）使用，建議使用最新版 Chrome / Firefox / Safari；行動版支援 iOS / Android / Microsoft，讓老師和學生用手機或平板創作與探索。

**內容：**
- 三步流程核心精神：免費、快速、低技術門檻
- 技術需求速查（Quick tech check）：
  - Web：任何電腦，含 Chromebook
  - 瀏覽器：最新版 Chrome / Firefox / Safari
  - Mobile：iOS / Android / Microsoft app
- 支援管道：
  - 社群論壇：`forum.edu.delightex.com`
  - Facebook 社群：Delightex Edu Community

> 「This step-by-step guide will take you through the three main steps to get started for FREE with Delightex Edu in the classroom.」

### Section 2｜入門手冊：Sign up → Create → Explore

**重點摘要：** 三步流程的具體操作細節。Sign up 階段在 `delightex.com/edu` 點擊 Sign up、建立教師帳號、選擇登入方式；Create 階段進入 Projects 頁面建立第一個 Project，從歡迎專案開始熟悉介面，並選擇 3D 場景、360° 圖片或 MERGE Cube 環境，再從 Library 拖放 3D 物件；Explore 階段提供四種探索模式：Play mode、Gyro mode、VR mode、AR mode。

**內容：**

| 步驟 | 動作 |
|------|------|
| **Sign up** | 1. 前往 `delightex.com/edu` → 點 Sign up 2. 選 Create an account as a teacher 3. 選擇登入方式 |
| **Create** | 1. 進入 Projects 2. 探索 welcome Project 3. 點 Create Project 4. 選擇 3D 環境 / 360° 圖片 / MERGE Cube 5. 從 Library 拖放 3D 物件，或在 Building 用 building blocks 自建 6. Upload 額外 3D 模型 / 圖片 / GIF / 影片 / 聲音 |
| **Explore** | Play mode / Gyro mode / VR mode / AR mode 四種探索方式 |

### Section 3｜教師 Checklist（5 步驟）

**重點摘要：** 教師 Checklist 把三步流程展開為 5 個可勾選的步驟 — 從免費 Basic 帳號（30 個 seats）開始，選擇性升級 Pro，建立第一個 class，邀學生加入 class（透過 class code），最後建立第一個 assignment。

**內容：**

| 步驟 | 細節 |
|------|------|
| 1. Sign up on `delightex.com/edu` | 註冊免費 Basic 教師帳號（30 seats：老師 + 最多 29 學生），開啟註冊信 |
| 2. Optional：升級 Pro | 線上購買或 quote + PO 流程，付費後取得 Pro 解鎖金鑰 |
| 3. 建立第一個 class | 左側選單 → Classes → Create class |
| 4. 邀學生加入 class | 取得 class code，分享給學生讓他們加入 |
| 5. 建立 assignment | 在 class 內 → Assignments → Create assignment |

### Section 4｜學生 Checklist（3 步驟）

**重點摘要：** 學生端的上手流程相對單純，分為 3 步 — 從電腦或手機建立學生帳號並輸入教師提供的 class code，進入 class，最後打開教師指派的 assignment 並閱讀指示後開始。

**內容：**
- 教師：把這份 checklist 印出來或轉給學生
- 學生 3 步驟：
  1. **Create your Delightex Edu account**（電腦 / 手機 / 平板都能註冊；輸入 class code；選一個記得住的登入方式）
  2. **Enter your class**（左側選單 → Classes → 進入你的 class）
  3. **Open your assignment**（在 class 內找到 assignment，**先讀指示再開工**）

> 「Make sure to read the assignments' instructions before getting to work!」

### Section 6｜鍵盤快捷鍵 Cheat Sheet

**重點摘要：** 平台提供完整的鍵盤快捷鍵 cheat sheet，分為 Desktop 與 Mobile 兩種環境、EDITOR 與 PLAY MODE 兩種模式。Desktop 快捷鍵涵蓋 Navigation（旋轉 / 縮放 / 平移視角）、Building（旋轉模式 / 群組 / 複製 / 貼上 / 對齊）、CoBlocks（複製 / 貼上整塊）、PLAY MODE（Flying / Walking / Orbit 三種攝影機控制）。Mobile 快捷鍵則對應簡化版。

**內容：**

**Desktop / EDITOR**

| 分類 | 快捷鍵 |
|------|--------|
| Navigation | 旋轉視角 / 縮放 / 平移視角 / 視野對齊物件 / 對齊滑鼠位置 / Zoom in/out |
| Building | 旋轉模式 / 群組 / 取消群組 / 選取全部 / 複製 / 貼上 / 複製並位移 / 取消選取 / Grid 吸附切換 |
| CoBlocks | 複製選取 CoBlock / 貼上 / 從當前 workspace 複製所有 CoBlocks |

**Desktop / PLAY MODE**

| 攝影機模式 | 操作 |
|------------|------|
| Flying | 前後左右移動 + 上升下降 |
| Walking | 前後左右移動 + 跳 |
| Orbit | 旋轉視角 / 縮放 / 平移 |

**Mobile**：對應簡化版（旋轉 / 縮放 / 傾斜 / 平移 / 選取 / 縮放 / 移動 / 物件檢查）

> 完整 cheat sheet 可在文件取得，或線上取得「Get printable version」。

## 四、核心主旨

> Delightex Edu 用 3 步流程（Sign up → Create → Explore）把 3D / VR / AR 帶進教室：教師建立免費 Basic 帳號與 class，學生用 class code 加入並打開 assignment；中間的 Project 創作則透過 5 種典型應用（360° tours / 虛擬展覽 / 互動故事 / 遊戲 / 模擬）、鍵盤快捷鍵 cheat sheet 與 CoBlocks 視覺化程式語言支援完成。
