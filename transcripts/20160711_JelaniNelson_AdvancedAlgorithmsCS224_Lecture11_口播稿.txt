這堂課完成三件事。第一是用 dual fitting 完整證明 weighted set cover 的 log n approximation。Greedy 演算法每次選 cost 除以新覆蓋元素數量最小的 set；邊做邊構造 dual solution：對每個被選的 set S，把它 cover 的每個新 element i 的 y_i 設為 cost(S) 除以 S 涵蓋的新 elements 數量，使得所有 i ∈ S 的 y_i 總和剛好等於 cost(S)。這時 dual 並不 feasible——每個 set 的 constraint 被違背的程度最多是 log n 倍。把所有 y_i 除以 log n 後變成 feasible，因此 greedy 的 primal cost ≤ log n 乘以 dual value ≤ log n 乘以離線 OPT，這就是 log n approximation 的證明。

第二件事是用 dual fitting 證明 unweighted vertex cover 的 2-approximation via primal-dual。Vertex cover 的 LP 是 minimize Σ x_v subject to x_u + x_v ≥ 1 for each edge。對未 covered edge 同時 raise x_u 與 x_v 直到其中一個為 1，把那個 vertex 加入 solution，重複直到所有 edges 被 cover。關鍵性質：每個被加入的 vertex 對應的 raise 最多 2 倍，因此 solution 大小 ≤ 2 乘以 dual value ≤ 2 乘以 LP optimal ≤ 2 乘以 integer optimal，這是 vertex cover 的 2-approximation 證明。

第三件事是引入 LP integrality gap 概念與 PTAS 框架。Integrality gap 是 LP optimal 與 integer optimal 的比，衡量 LP relaxation 的鬆度。對 set cover，這個 gap 是 Ω(log n)，代表 LP-based 方法對 set cover 最多就到 log n。當 LP 不夠時，PTAS 是另一條路：對任何 ε > 0 給出 1+ε approximation 在多項式時間。對 knapsack problem 構造 PTAS，把 items 分成 big（weight 大於 εC）與 small（weight ≤ εC）：對 big items 用 DP 求 exact optimal subset，剩餘 capacity 用 greedy 處理 small items，整體達到 1-ε approximation。

整堂課展示的核心思想在於：dual fitting 把 approximate CS 與 offline 構造結合，是 LP duality 在 approximation algorithms 上的標準工具之一；PTAS 則是繞過 LP 限制的另一條路，對某些問題（如 knapsack）已知可達。下節課會進一步深入 FPTAS 與另一個 problem 的 FPTAS 構造，把這套 approximation algorithms 設計方法論推廣到更廣的問題類型，並指出 integrality gap 與 computational complexity 的深沉關係——若某 problem 有 instance complexity 證明，PTAS 通常不存在；反之若有 PTAS，通常 LP 也有充分緊密 bound，這是 LPC 與 complexity 理論接軌的關鍵點之一。
