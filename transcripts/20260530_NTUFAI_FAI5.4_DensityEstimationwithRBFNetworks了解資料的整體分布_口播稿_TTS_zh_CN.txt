这段节目由林轩田教授介绍如何使用径向基底函数网路（RBF Network）来做密度估计（Density Estimation），这是密度估计系列中最抽象、也最具模型观点的一种做法。

节目一先复习机率密度函数（PDF）的概念：对一个连续随机变数，PDF 给出落在某个位置附近的机率密度。实务上常以直方图估计，但直方图的问题是切 bin 宽度会严重影响形状，邻近 bin 的差异可能被过度放大或抹平。教授点出，更好的做法是把密度视为若干「高斯形的山丘」加总而成，这正是 RBF Network 的物理图像。

RBF Network 的结构与传统神经网路相似，但中间层不是 sigmoid 或 tanh 函数，而是径向基底函数（Radial Basis Function）。教授先介绍 RBF 的两个关键字：Basis Function 是构成整体形状的基本元件；Radial 描述的是「以某个中心 μ 为基准，距离越远强度越弱」的形状，最典型的就是高斯函数。把 K 个高斯基底函数各自乘上一个权重 β，再加上一个偏置项，线性组合起来就是 RBF Network 的输出公式。当有监督资料时，β 与 μ 可以直接学；本节重点放在无监督情境下的做法。

最简单的无监督做法叫做朴素密度估计（Naive Density Estimation）：假设每一笔观测到的 X 都来自一个独立的、以该 X 为中心的高斯分布，且每个高斯的权重相等（都是 1/N）。把所有高斯加起来，就得到整体密度的估计值。这种做法后来被称为核密度估计（Kernel Density Estimation, KDE）。

然而，KDE 把每一笔 X 都当成一个独立高斯，过于精细且容易过拟合。教授提出更务实的做法：先用 K-Means 把资料分成 K 群，再以每群的中心作为 RBF 的中心 μ，每群的权重 β 则依群内资料比例设定。这种 hybrid 做法既保留了 RBF 的几何直觉，又避免了 KDE 过度拟合的问题。

节目最后比较 RBF 与一般神经网路的差异：一般神经网路中间层是 sigmoid 等内积型函数，RBF 则是距离型函数；两者的输出层都是线性组合。教授也点出 RBF Network 在异常侦测上的应用：当一个新样本落在 RBF 估计出的低密度区域，就能合理怀疑它是异常点。

整段节目的核心讯息是：RBF Network 把密度估计问题转化为「以几个代表性的高斯中心为基底，线性组合出整体分布」的模型问题；KDE 是最直观的特例，K-Means + RBF 是更具代表性的改良方案——这条思路展示了非监督式学习如何用浅层模型逼近复杂分布。
