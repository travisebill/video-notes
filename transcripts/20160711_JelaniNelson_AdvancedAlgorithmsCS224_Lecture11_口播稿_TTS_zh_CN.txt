这堂课完成三件事。第一是用 dual fitting 完整证明 weighted set cover 的 log n approximation。Greedy 演算法每次选 cost 除以新覆盖元素数量最小的 set；边做边构造 dual solution：对每个被选的 set S，把它 cover 的每个新 element i 的 y_i 设为 cost(S) 除以 S 涵盖的新 elements 数量，使得所有 i ∈ S 的 y_i 总和刚好等于 cost(S)。这时 dual 并不 feasible——每个 set 的 constraint 被违背的程度最多是 log n 倍。把所有 y_i 除以 log n 后变成 feasible，因此 greedy 的 primal cost ≤ log n 乘以 dual value ≤ log n 乘以离线 OPT，这就是 log n approximation 的证明。

第二件事是用 dual fitting 证明 unweighted vertex cover 的 2-approximation via primal-dual。Vertex cover 的 LP 是 minimize Σ x_v subject to x_u + x_v ≥ 1 for each edge。对未 covered edge 同时 raise x_u 与 x_v 直到其中一个为 1，把那个 vertex 加入 solution，重复直到所有 edges 被 cover。关键性质：每个被加入的 vertex 对应的 raise 最多 2 倍，因此 solution 大小 ≤ 2 乘以 dual value ≤ 2 乘以 LP optimal ≤ 2 乘以 integer optimal，这是 vertex cover 的 2-approximation 证明。

第三件事是引入 LP integrality gap 概念与 PTAS 框架。Integrality gap 是 LP optimal 与 integer optimal 的比，衡量 LP relaxation 的松度。对 set cover，这个 gap 是 Ω(log n)，代表 LP-based 方法对 set cover 最多就到 log n。当 LP 不够时，PTAS 是另一条路：对任何 ε > 0 给出 1+ε approximation 在多项式时间。对 knapsack problem 构造 PTAS，把 items 分成 big（weight 大于 εC）与 small（weight ≤ εC）：对 big items 用 DP 求 exact optimal subset，剩余 capacity 用 greedy 处理 small items，整体达到 1-ε approximation。

整堂课展示的核心思想在于：dual fitting 把 approximate CS 与 offline 构造结合，是 LP duality 在 approximation algorithms 上的标准工具之一；PTAS 则是绕过 LP 限制的另一条路，对某些问题（如 knapsack）已知可达。下节课会进一步深入 FPTAS 与另一个 problem 的 FPTAS 构造，把这套 approximation algorithms 设计方法论推广到更广的问题类型，并指出 integrality gap 与 computational complexity 的深沉关系——若某 problem 有 instance complexity 证明，PTAS 通常不存在；反之若有 PTAS，通常 LP 也有充分紧密 bound，这是 LPC 与 complexity 理论接轨的关键点之一。
