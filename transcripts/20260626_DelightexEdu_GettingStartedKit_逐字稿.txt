As easy as 1, 2, 3!

This step-by-step guide will take you through the three main steps to get started for FREE with Delightex Edu in the classroom.

Sign up → Create → Explore

Quick tech check:
- The Delightex Edu web app works on any computer (including Google Chromebooks). We recommend using the latest version of Google Chrome, Firefox or Safari.
- The Delightex Edu mobile app for iOS, Android and Microsoft lets you create and explore your creations on a smartphone or tablet.

Need support?
- forum.edu.delightex.com
- Delightex Edu Community on Facebook

1. Sign up
1. Go to delightex.com/edu and click Sign up to create your Delightex Edu account.
2. Create an account as a teacher.
3. Choose how you'll log into your Delightex Edu account.

2. Create
- Go to Projects (where your creations are stored)
- Start by exploring the welcome Project
- Click Create Project
- Choose your first scene to build in:
  - 3D environment (viewable in VR and AR)
  - 360° image (viewable in VR)
  - MERGE Cube add-on
- Drag and drop 3D objects from Library
- Build your own using building blocks under Building
- Go to Upload to add 3D models, images, GIFs, videos, sounds

3. Explore (4 modes)
- Play mode: explore a Project and move around on any device
- Gyro mode: use tablet/smartphone gyroscope to look around
- VR mode: ClassVR, Pico headsets, or mobile VR (smartphone + VR-compatible)
- AR mode: AR-compatible tablet/smartphone + mobile app

Teacher checklist (5 steps)
1. Sign up on delightex.com/edu → Basic account (30 seats for you + up to 29 students)
2. Optional: Upgrade to Delightex Edu Pro
3. Create your first class (left menu → Classes → Create class)
4. Invite students to your class (share the class code)
5. Create an assignment (in your class → Assignments → Create assignment)

Student checklist (3 steps)
1. Create your Delightex Edu account (computer → delightex.com/edu/Register; phone/tablet → app)
2. Enter your class (left menu → Classes)
3. Open your assignment (read instructions before starting)

Desktop shortcuts (Editor + Play mode)
- Editor Navigation: Rotate camera view / Center in on selected object / Zoom in/out / View whole scene / Move camera view / Zoom in / Zoom camera to mouse position / Zoom out
- Editor Building: Rotation mode / Turn grid snapping on/off / Group/ungroup selected objects / Select all objects / Copy selected object(s) / Paste object(s) / Duplicate and move / Deselect object(s)
- Editor CoBlocks: Copy selected CoBlock(s) / Paste CoBlocks / Copy all CoBlocks from current workspace
- Play mode Flying camera: Move forward / Move up / Move left / Move down / Move backward / Move right
- Play mode Walking camera: Move forward / Jump / Move left / Move backward / Move right
- Play mode Orbit camera: Rotate camera view / Zoom in / Move camera view / Zoom out / Zoom in/out

Mobile shortcuts (Editor + Play mode)
- Editor Navigation: Rotate camera view left/right / Zoom in/out / Tilt camera view up/down / Move camera view
- Editor Building: Select object / Scale object / Move object / Open object inspector
- Play mode Object interaction
- Walking/Flying/Fixed camera: Turn camera / Move forward
- Orbit camera: Rotate camera view left/right / Zoom in/out / Tilt camera view up/down / Move camera view
